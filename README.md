# unity-health-backend
Unity Health Backend is a robust Health Record management system built with C# and EF Core. Optimized for PostgreSQL, it streamlines medical workflows by managing referrals, client/patient records, and doctor scheduling. Designed for high performance and data integrity, It powers modern healthcare with easy scheduling and smooth medical care.


# 🚀 Features
Patient Management: Comprehensive tracking of patient details and history.

Doctor Coordination: Manage medical staff and availability.

Appointment Scheduling: Seamless booking and coordination system.

Referral System: Efficiently track and manage medical referrals.

Client Records: Manage billing or primary contact information.

# 🛠️ Tech Stack
Language: C# (.NET 8/9)

Database: PostgreSQL

ORM: Entity Framework Core

API Documentation: Swagger/OpenAPI
